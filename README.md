# Customer-Shopping-Behavior-Analysis

This project focuses on analyzing customer shopping behavior using Python, SQL, and BI tools. The goal is to extract meaningful insights from raw transactional data, perform exploratory data analysis (EDA), apply feature engineering, and present insights through interactive dashboards and reports. This project is designed to be recruiter-friendly and demonstrates end-to-end data analytics skills.

# **Dataset**

Source: Customer shopping behavior dataset (CSV format)

Content Includes:

Customer ID

Age, Gender

Category & Item Purchased

Purchase Amount

Purchase Frequency

Discount Applied

Payment Method

Location / Season (if available)


# **Tools & Technologies**

**Programming:** Python (Pandas, NumPy, Matplotlib)

**Database:** MySQL / PostgreSQL / SQL Server

**Visualization & BI:** Power BI

**Reporting & Presentation:** Markdown (README), Canva (PPT)

**Environment:** Jupyter Notebook

# **Project Workflow / Steps**



**1️. Data Loading**

--> Loaded CSV dataset into Python using Pandas

--> Verified schema, data types, and basic structure




**2️. Data Cleaning**

--> Handled missing values

--> Corrected data types

--> Standardized categorical values

--> Removed duplicates






**3️. Exploratory Data Analysis (EDA)**

--> Distribution of purchase amounts

--> Customer demographics analysis

--> Category-wise and frequency-based analysis

--> Discount usage patterns

**4️. Feature Engineering**

--> Purchase frequency mapping

--> Discount rate calculation

--> Customer segmentation features

--> Aggregated KPIs for analysis

**5️. SQL Analysis**

--> Imported cleaned data into SQL database

--> Executed queries such as:

--> Top-selling categories and items

--> Discount impact analysis

--> Customer purchase frequency analysis

--> Category-wise performance metrics

**6️. Data Visualization (Python)**

--> Bar charts, pie charts, and trend plots using Matplotlib

--> KPI-focused visualizations aligned with BI dashboards

**7️. Power BI Dashboard**

--> Interactive dashboards with:

--> KPIs & summary cards

--> Category-wise sales

--> Discount impact analysis

--> Customer behavior trends

**8️. Reporting & Presentation**

--> Documented insights and results

--> Created a presentation using Canva

--> Added dashboards and key findings

--> Identified high-performing categories and products

# **Conclusion**

This Customer Shopping Behavior Analysis project demonstrates how raw customer transaction data can be converted into meaningful insights through systematic analysis. Using Python and SQL, the data was cleaned, explored, and analyzed to uncover key patterns related to purchase frequency, category-wise demand, popular items, and discount usage. Visualizations played a crucial role in highlighting trends and making complex data easy to understand.

The insights derived from this analysis can support better business decisions such as targeted marketing, effective discount strategies, and improved inventory management. Overall, the project showcases an end-to-end data analytics workflow and reflects the practical application of analytical tools to understand customer behavior and drive data-informed strategies.
